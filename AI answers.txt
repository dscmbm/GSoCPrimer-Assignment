Q.6 Write a Python function that calculates the factorial of a
     given number. Using recursion?
Ans: def factorial(n):
        if n==0 or n==1:
            return 1
        else:
            return n*factorial(n-1)

Q 10 Write a Python function that removes duplicates from a given 
    list and returns a new  list with unique elements.

Ans: def duplicate(I_list):
        unique=[]
        for i in I_list:
            if i not in unique:
                unique.append(i)
        return unique
